# Tutor Report Generator

Автогенератор резюме тьюторской встречи для Zerocoder — веб-сервис на базе Claude API, который превращает транскрипцию встречи в структурированный отчёт для вип-менеджера за секунды.

---

## Проблема

После каждой тьюторской встречи с VIP-студентом тьютор вручную пишет резюме для вип-менеджера: персональные договорённости, рекомендации по программе, следующие шаги, важные моменты. При 25 встречах в месяц это ~10 минут после каждой — и нередко этих 10 минут нет, встречи идут одна за другой, и к окончанию серии встреч уже сложно вспомнить детали первых разговоров. Кроме того, такой отчет сокращает время на подготовку к следующей тьюторской с этим студентом: быстро вспомнить, с какими ожиданиями приходил человек, какие были договоренности. 

**Tutor Report Generator** закрывает эту рутину: загрузил транскрипцию из корп аккаунта → получил готовый отчёт → скопировал и отправил вип-менеджеру.

---

## Что умеет
- **Загрузка файлов** — поддержка drag & drop и загрузки `.txt` / `.docx`
- **Анализирует транскрипцию** встречи через Claude API (claude-sonnet-4-6)
- **Генерирует структурированное резюме** для вип-менеджера:
  - Основные тезисы встречи
  - Квалификация студента по уровню соответствия и подготовки к программе 🟢 Крутыш / 🟡 Середнячок / 🔴 Ред флаг с пояснением
  - Список договорённостей со студентом и задачи для тьютора и/или вип-менеджера
- **Кнопка «Скопировать»** — результат сразу готов к отправке
- **Загрузочный экран** с забавными фразами
- **Анимация** Конфетти при зелёном студенте, red flag анимация при красном
- **Счётчик сэкономленного времени** в шапке — мотивирует использовать
- **Автосохранение транскрипций** в папку `raw-data/` (база транскрибаций, чтобы при необходимости можно было быстро погрузиться в детали предыдущей встречи)

## Стек

| Слой | Технология |
|---|---|
| Интерфейс | HTML / CSS / Vanilla JavaScript |
| AI-анализ | Claude API (`claude-sonnet-4-6`) |
| Хранилище | Локальная папка `raw-data/` |
| Зависимости | [mammoth.js](https://github.com/mwilliamson/mammoth.js) для `.docx` |

Никаких фреймворков, никакого бэкенда. Один файл `index.html`, открывается локально в браузере.

## Быстрый старт

### 1. Клонировать репозиторий

```bash
git clone https://github.com/flashkate/tutor-report-generator.git
cd tutor-report-generator
```

### 2. Получить Claude API Key

Зарегистрироваться на [console.anthropic.com](https://console.anthropic.com), создать API-ключ в разделе **API Keys**.

### 3. Запустить

Открыть `index.html` в браузере. В правом верхнем углу нажать **⚙ Настройки**, вставить API-ключ — он сохранится в `localStorage`.

### 4. Использовать

1. Вставить текст транскрипции в поле (или загрузить файл `.txt` / `.docx`)
2. Нажать **Сгенерировать**
3. Скопировать готовое резюме кнопкой **Скопировать**

## Формат резюме

Пример сгенерированного отчёта:

```
📝 **Мини-резюме тьюторской**
Андрей — студент программы AI-студия, без технического бэкграунда в IT и программировании. Пришёл с целью сменить текущий заработок на работу промпт-инженера/владельца AI-студии, мотивирован на быструю монетизацию.

🟡 **Середнячок**
Студент мотивирован и задаёт конструктивные вопросы, однако нет IT-бэкграунда — рекомендовано предобучение. Дополнительно есть технические трудности со стартом: нет доступа к Telegram и учебной платформе из-за корпоративных ограничений на VPN.

🤝 **Договорённости**
- Андрею обязательно проходить предобучение (основы Python, нейросети, no-code) — ориентировочно 3–4 недели.
- Первоочередная задача до старта обучения — настроить Telegram и доступ к учебной платформе (VPN/прокси); при необходимости старт можно перенести — сигнализировать вип-менеджеру.
- После встречи выслать презентацию и ссылку на вебинар по ценообразованию.
```
