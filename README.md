# Tutor Report Generator

Автогенератор резюме тьюторской встречи для Zerocoder — веб-сервис на базе Claude API, который превращает транскрипцию встречи в структурированный отчёт для вип-менеджера за секунды.

---

## Проблема

После каждой тьюторской встречи с VIP-студентом тьютор вручную пишет резюме для вип-менеджера: персональные договорённости, рекомендации по программе, следующие шаги, важные моменты. При 25 встречах в месяц это ~10 минут после каждой — и нередко встречи идут одна за другой, и к вечеру уже сложно вспомнить детали первых разговоров.

**Tutor Report Generator** закрывает эту рутину: вставил транскрипцию → получил готовый отчёт → скопировал вип-менеджеру.

---

## Что умеет

- **Анализирует транскрипцию** встречи через Claude API (claude-sonnet-4-6)
- **Генерирует структурированное резюме** для вип-менеджера:
  - Основные тезисы встречи
  - Квалификация студента по светофору 🟢 / 🟡 / 🔴 с пояснением
  - Список договорённостей со студентом
  - Задачи для тьютора и/или вип-менеджера
- **Кнопка «Скопировать»** — результат сразу готов к отправке
- **Анимации по квалификации** — конфетти при 🟢, red flag при 🔴
- **Загрузочный экран** с забавными фразами и случайной цитатой про тьюторство
- **Счётчик сэкономленного времени** в шапке — мотивирует использовать
- **Автосохранение транскрипций** в папку `raw-data/` с датой и временем
- **Загрузка файлов** — поддержка drag & drop и загрузки `.txt` / `.docx`

## Стек

| Слой | Технология |
|---|---|
| Интерфейс | HTML / CSS / Vanilla JavaScript |

| Хранилище | Локальная папка `raw-data/` |

Никаких фреймворков, никакого бэкенда. Один файл `index.html`, открывается локально в браузере.

## Быстрый старт

### 1. Клонировать репозиторий

```bash
git clone https://github.com/flashkate/tutor-report-generator.git
cd tutor-report-generator
```

### 2. Получить Claude API Key

Зарегистрироваться на [console.anthropic.com](https://console.anthropic.com), создать API-ключ в разделе **API Keys**.

### 3. Запустить

Открыть `index.html` в браузере. В правом верхнем углу нажать **⚙ Настройки**, вставить API-ключ — он сохранится в `localStorage`.

### 4. Использовать

1. Вставить текст транскрипции в поле (или загрузить файл `.txt` / `.docx`)
2. Нажать **Сгенерировать**
3. Скопировать готовое резюме кнопкой **Скопировать**
