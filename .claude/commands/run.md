Открой сервис в браузере командой:

```bash
open index.html
```

После открытия убедись, что:
1. Страница загрузилась без ошибок в консоли (DevTools → Console)
2. Счётчик в шапке отображается корректно
3. Поле для вставки транскрипции и кнопка «Сгенерировать» видны
4. Если API key уже сохранён — блок настроек не должен показывать предупреждение

Если открылась пустая страница — проверь, что файл `index.html` существует в текущей папке.
